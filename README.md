## ♰ MANIFESTO
> *"In the silence of the graveyard, the order is absolute."*

**CRUCIFIX** — это военизированная графическая оболочка, созданная для тех, кто ищет эстетику строгости и глубокого погружения. Вдохновленная образами темного аниме и милитари-готики, она превращает рабочее пространство в цифровой штаб. Здесь нет места лишнему — только функционал, облеченный в сталь и кровь.

## ♰ THE RITE (Инициализация)
Для развертывания системы в вашем секторе необходимо провести ритуал компиляции:

```bash
# Доступ к исходному коду
git clone https://github.com/4ERNIY-LOSOS/Crucifix.git

# Вход в защищенный периметр
cd Crucifix

# Запуск протокола установки
./scripts/deploy.sh
```
